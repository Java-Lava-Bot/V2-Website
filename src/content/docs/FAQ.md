# Frequently Asked Questions

## General Questions

### What is Java Lava?

Java Lava is a powerful Discord bot designed to enhance your server experience with moderation tools, fun commands, and community features. It is developed by PhillsPhanbh3_the_bot_dev#0000.

### How do I add Java Lava to my server?

Click the [Invite Bot](https://discord.com/oauth2/authorize?client_id=1305190785536360519) link and select your server. Make sure you have "Administrator" permissions.

### Is Java Lava free to use?

Yes! Java Lava is free to use. We also offer premium features for enhanced functionality.

### What version is Java Lava currently on?

Java Lava is currently on version 1.20! Check our [version history](/Updates/) for detailed changelog information.

## Technical Issues

### Java Lava is offline, what do I do?

If Java Lava appears offline, it might be:

- Experiencing temporary downtime
- Having connection issues

Check our [Discord server](https://discord.gg/6tF3UFWA6F) and [Status Page](https://javalava.statuspage.io/) for status updates.
