### What was in version 1.01?

There was only 1 thing in version 1.01 and it was a /mass-unban command only usable by guild owners in the event of a server getting nuked and you wanted to restore your server due to the amount of bans that had occurred from the nuke.

## Why was this command added in a emergency?

1. I gotten informed about a discord server that gotten nuked and I decided to help make a mass unban command on the bot for that specific server and made it's access to anywhere

2. I also woke up to the notification about the issue which I was very happy to help the server with the issue. It took me around 45 minutes to code the update and to double check all the code to make sure it was correct!
