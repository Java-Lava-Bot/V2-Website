# Java Lava Version 1.00

## We have released version 1.00 the 1st ever version of Java Lava

---

### Welcome to Java Lava version 1.00

!!!
Java Lava version 1.00 released on December 22nd 2024
!!!

You may ask what was in the 1st ever version of Java Lava? A lot. A whole heck of a lot.

1. We added in a FAMILY of automod commands! We added in with this family of automod commands /automod flagged-words, /automod spam-messages, /automod mention-spam, and last but not least within this current update is /automod keyword. The automod system we had for version 1 of Java Lava was let's just say, not the best at all, I PhillsPhanbh3 knew NOTHING about coding except for some YouTube tutorials. The YouTube tutorials I used were from MrJAwesome https://www.youtube.com/@MrJAwesomeYT and his team of coding helpers in his server is super helpful, one of his helpers is one of the developers of the bot! Now back onto the actual update.

2. We added in a /say command that had gotten removed a few updates later and fully phased out of all bots when version 1.08 came around. That command was where you could send messages through the command without ANYONE knowing who sent it, this sparked a lot of issues within servers I was staff for because people were constantly abusing it, I was going to make it require admin or manage server perms but I decided it was better to fully take it out of the bot.

3. We added in a /poke command which is a little fun command where you can poke the bot for being a bad bot and causing itself to crash more times than I can count so it gotten poked a ton of times.

4. We added in a /clicker command where you can click a button and the bot will edit the message for the amount of times that the command would go up, I almost caused the bot to get rate limited one time I pressed it like 400 times without issues until I checked my terminal and saw Discord API warning you are about to get rate limited, this wasn't the only time that the bot almost got rate limited though.

5. Lastly we added in a /ping command which that command just said pong with no latency reply somehow, that was super confusing but then like 3 updates later I got a real /ping command into the bot with latency.

---

I really had fun making the 1st version of Java Lava and I knew there was going to be more to come! and I was right.
