# Java Lava Version 1.04

## Version 1.04

!!!
Java Lava Version 1.04 has released on January 19th 2025
!!!

### What is new about this update?

#### New Moderation Commands
1. This new update is about moderation!

2. We have added /timeout, a /gag-a-member command (another version of the /timeout command), /kick, /ban, /unban commands - the normal moderation commands on the bot, also decided to add in /ping-latency onto the bot as well like 45 minutes after the announcement of this update

#### Website Updates
3. I have made a Google Sites website page for the bot which turned from https://sites.google.com/view/javalavabot/home which was deactivated, we turned it into https://javalava.discloud.app/

#### Infrastructure
4. We are also looking into a hosting service for Java Lava due to how bad the current hosting is.
