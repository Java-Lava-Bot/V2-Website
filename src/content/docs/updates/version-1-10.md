# Version 1.10 Update Notes

## New Features

- **Advanced Role Management**: Create role hierarchies and set permissions in bulk
- **Custom Command Builder**: Create your own commands with a visual builder
- **Voice Effects**: Apply effects to your voice in voice channels
- **Reaction Roles 2.0**: Completely redesigned with more options and better UI

## Improvements

- Bot response time improved by 35%
- Better handling of rate limits
- Enhanced error messages with helpful troubleshooting tips
- Command suggestions when misspelled

## Bug Fixes

- Fixed an issue where the music player would stop after 30 minutes
- Fixed leaderboard pagination
- Role persistence now works correctly after server outages
- Welcome messages now properly show custom images

## Other Changes

- Updated dependencies for better security
- New documentation site launched
- Premium tier pricing adjustment
